# Average Volatility Dimensioning (AVD): Dimension Reduction Technique for Multivariate Time Series
Reducing the features while preserving the inner dynamics and correlations of a dataset is a challenging task. This is particularly evident for non-linear and time-dependent processes. We propose a novel dimension reduction technique, Average Volatility Dimensioning (AVD), capable of capturing the behavior between features and reducing the multiple dimensions to an univariate signal. We compare this dimension reduction technique against state-of-the-art methods (PCA, MDS, DyCA) and evaluate its usefulness on three datasets and machine learning tasks. It is shown that AVD encapsulates the dynamics of the data best for the given datasets and can increase the prediction accuracy of classifiers compared to the original dataset.
